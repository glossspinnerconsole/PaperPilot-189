Online demo

输入调研需求，生成检索计划和候选论文。

这个在线体验运行在 Cloudflare Workers 上，使用服务端 LLM 生成英文检索式、研究问题和纳入标准，再查询公开论文 metadata API 并生成可下载的轻量 Markdown/HTML 报告。完整 PDF、Evidence Ledger、Obsidian Wiki 和完整 PaperPilot CLI workflow 需要在本地运行。

轻量在线体验

这里不会运行完整本地 Python CLI workflow，也不会抽取 PDF 全文。结果基于公开 API metadata/abstract，适合快速形成检索计划和候选列表。

等待检索

提交后这里会显示研究问题、英文检索式、候选论文和报告下载按钮。
